A quick, no-frills terminal Snake game

🎮 Ready to play?

Click "Run Locally" to start the game in your terminal

Score: 0

How to Play

⬆️ W / Arrow Up - Move Up
⬇️ S / Arrow Down - Move Down
⬅️ A / Arrow Left - Move Left
➡️ D / Arrow Right - Move Right
❌ Q / Ctrl+C - Quit
💡 How to run:
  1. Make sure you have Python 3 installed
  2. Clone or download the repository
  3. Open terminal and run: python ozzsnake.py
  4. Use keyboard controls to move the snake
